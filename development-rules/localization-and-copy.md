# Localization And Copy

- No hardcoded user-facing copy in JSX.
- English, Polish, and Ukrainian must remain supported.
- Add new copy to all locale resources in the same change.
- Route labels, aria labels, button text, form labels, empty states, and error text are user-facing copy.
- Domain labels should use typed copy keys rather than scattered strings.
- Do not use machine translation blindly for critical product or error text.
