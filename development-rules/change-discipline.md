# Change Discipline

- Keep each iteration independently runnable.
- Stay inside the iteration scope.
- Do not add TODO placeholders unless the reason is explicit and actionable.
- Do not leave dead files, unused CSS, commented-out code, or debug logs.
- Do not reset cadence during normal digest refresh.
- After each iteration run lint, typecheck, tests, and build.
- Document intentional deferrals in the iteration notes or PR body.
